# SSA Name Popularity Visualizer

A static browser app for exploring Social Security baby name popularity from 1980 to 2025.

Features:

- National data from `names/`
- State data from `namesbystate/`
- Combined, female, male, or separate gender lines
- Comma-separated name spellings added together year by year

## Run Locally

```bash
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000`.
